About This Project

The Yapi Linguistics Team aims to create an engaging, non-gatekeeping linguistic classification system that isn't as closed as Glottolog.
We welcome any ideas—especially from native speakers—about how languages are grouped. We are very open-minded when it comes to change.

Together we can grow this community. (GitHub link)
